## **阶段 0：建立基础认知**

目标：理解金融市场、资产和量化投资的基本概念。

### **0.1 了解金融市场**

* 金融市场类型：

  1. **股票市场**：公司股份交易。
  2. **债券市场**：借贷行为，固定收益。
  3. **期货市场**：标准化合约，约定未来价格买卖。
  4. **外汇市场**：货币兑换交易。
  5. **衍生品市场**：期权、互换、结构化产品。
* 关键概念：

  * **收益率**（return）
  * **风险**（risk / volatility）
  * **资产组合**（portfolio）
  * **分散投资**（diversification）

### **0.2 金融基础知识**

* **股票基本知识**：

  * 股价、成交量、市值、PE、PB
* **债券基础**：

  * 票面利率、到期收益率、久期
* **投资指标**：

  * 夏普比率（Sharpe Ratio）
  * 最大回撤（Max Drawdown）
  * 年化收益率

### **0.3 为什么做量化投资**

* 人类决策容易受情绪影响
* 量化投资通过数据和算法做决策
* Python 和数学可以把策略自动化

## **阶段 1：Python + 金融数据分析**

目标：学会用 Python 获取金融数据、做简单分析。

### **1.1 学习用 Python 获取数据**

* 推荐库：

  * `pandas`（数据处理）
  * `numpy`（数值计算）
  * `matplotlib` / `plotly`（可视化）
  * `yfinance`（获取股票历史数据）
  * `tushare`（国内股票数据）
* 入门练习：

```python
import yfinance as yf

# 下载苹果公司过去1年的数据
data = yf.download("AAPL", period="1y")
print(data.head())
```

### **1.2 学习金融数据分析**

* 计算收益率

```python
data['return'] = data['Adj Close'].pct_change()
```

* 绘制收益率曲线

```python
import matplotlib.pyplot as plt
data['Adj Close'].plot()
plt.show()
```

* 简单统计：

```python
print(data['return'].mean())
print(data['return'].std())
```

### **1.3 学会做简单策略**

* 均线策略（Moving Average）

```python
data['MA10'] = data['Adj Close'].rolling(10).mean()
data['MA50'] = data['Adj Close'].rolling(50).mean()
```

* 生成交易信号：

```python
data['signal'] = 0
data.loc[data['MA10'] > data['MA50'], 'signal'] = 1  # 买
data.loc[data['MA10'] < data['MA50'], 'signal'] = -1 # 卖
```
